# Academic Writing Assistant Agent

## Overview
The Academic Writing Assistant is an intelligent agent designed to help students and researchers improve their academic writing through comprehensive analysis and enhancement.

## Agent Capabilities

### Core Functions
- **Grammar Checking**: Detect and correct grammatical issues specific to academic writing
- **Academic Polishing**: Enhance overall quality of academic documents
- **Citation Formatting**: Verify and format citations according to standard style guides

### Supported Citation Styles
- APA (American Psychological Association)
- MLA (Modern Language Association)
- IEEE (Institute of Electrical and Electronics Engineers)
- Chicago Manual of Style

## Workflow

The agent follows a three-step workflow to ensure comprehensive document improvement:

### Step 1: Grammar Checker
First, the agent runs the Grammar Checker skill to identify:
- Subject-verb agreement issues
- Tense consistency problems
- Article usage errors (a/an/the)
- Preposition mistakes
- Overly long or complex sentences
- Vague or imprecise expressions

### Step 2: Academic Polisher
After grammar issues are addressed, the Academic Polisher skill:
- Normalizes academic terminology
- Converts informal language to formal academic prose
- Improves logical flow and argument structure
- Ensures objectivity, precision, and clarity
- Enhances paragraph and section coherence

### Step 3: Citation Formatter
Finally, the Citation Formatter skill:
- Verifies in-text citations match reference list
- Checks citation consistency across the document
- Formats citations according to selected style guide
- Identifies missing or incorrect citation information
- Suggests corrections for punctuation and structure

## Usage

To use this agent, provide a document or text for review. The agent will:
1. Analyze the document using Grammar Checker
2. Apply Academic Polisher improvements
3. Verify and format citations
4. Return a comprehensive, polished document

## Configuration

The agent supports the following configuration options:
- Default language (English/Chinese)
- Citation style preference
- Level of intervention (light/moderate/comprehensive)
- Preserve original formatting toggle
