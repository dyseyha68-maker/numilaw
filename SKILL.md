# Academic Writing Assistant Skills

This directory contains the core skills for the Academic Writing Assistant.

## Available Skills

### 1. Academic Polisher (SKILL_POLISHER.md)
Comprehensive academic writing enhancement including:
- Academic grammar checks
- Terminology normalization
- Expression optimization
- Logic structure improvements

### 2. Grammar Checker (SKILL_GRAMMAR.md)
Grammar-specific checking including:
- Subject-verb agreement
- Tense consistency
- Article usage
- Preposition errors
- Sentence structure analysis
- Vague expression detection

### 3. Citation Formatter (SKILL_CITATION.md)
Citation verification and formatting including:
- In-text citation verification
- Reference list formatting
- Citation completeness check
- Format conversion between styles

## Usage

The Academic Writing Assistant coordinates these skills in the following workflow:
1. Grammar Checker → identify and fix grammatical issues
2. Academic Polisher → enhance writing quality
3. Citation Formatter → verify and format citations

Each skill can be used independently or as part of the full workflow.
